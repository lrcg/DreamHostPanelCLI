DreamHostPanelCLI
=================

Purpose
-------

An automatable command-line interface (CLI) interface for DreamHost's
custom Panel written in Perl.
 
Status
------

Incomplete. Can login, find domains and tasks, and forms on the Panel
pages for those tasks.
